# Movie API

 ### Ruby Version
 Ruby 3.0.0

 Rails 6.1.3

 ### Configuration
1. `git clone git@github.com:Kathybui732/movie_api.git`
2. `cd movie_api`
3. `bundle install`
4. `rake db:{create,migrate}`
5. `bundle exec rake import` to seed your database
6. `bundle exec rspec` to run the test suite
